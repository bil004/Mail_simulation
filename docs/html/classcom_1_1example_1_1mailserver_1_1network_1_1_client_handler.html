Detailed Description

Handles communication with a single client in a separate thread.

This class is responsible for processing requests from a connected client. It reads commands from the client's input stream, performs the requested actions (like sending, receiving, or deleting emails), and sends back a response. Each ClientHandler instance runs on its own thread, allowing the server to handle multiple clients concurrently.
