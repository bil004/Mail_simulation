Detailed Description

A multi-threaded server for handling client connections.

This class sets up a ServerSocket to listen for incoming client connections on a specified port. For each connection, it creates a new ClientHandler and submits it to a thread pool for concurrent processing.

The main execution logic for the server thread.

It creates a thread pool and a server socket, then enters a loop to accept client connections. Each new connection is passed to a ClientHandler in the thread pool.

Stops the server gracefully.

Sets the running flag to false and closes the server socket to interrupt the accept loop. The thread pool is shut down in the run method's finally block.
