Detailed Description

A simple multi-threaded server for handling client connections.

This class sets up a ServerSocket to listen for incoming client connections on a specified port. For each connection, it creates a new ClientHandler and submits it to a thread pool for concurrent processing.

Deprecated
This class is not actively used in the main application. The server logic is handled directly within the com.example.mailserver.MailServer class.
