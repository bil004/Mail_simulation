Client-Server Email System

Introduction

Welcome to the documentation for the Programming 3 project. The system is composed of two main modules:

  • MailServer: Manages data persistence (JSON) and multiple connections.
  • MailClient: User interface for managing emails (Send, Receive, Reply, Delete).

Architecture

The project uses a Model-View-Controller (MVC) architecture and communication based on TCP sockets with a custom text-based protocol.

Technologies Used

  • JavaFX for the graphical user interface.
  • GSON for data serialization.
  • ThreadPool (ExecutorService) for server-side multithreading.
