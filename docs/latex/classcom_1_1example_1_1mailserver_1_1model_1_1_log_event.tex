\doxysubsection{Detailed Description}
Represents a log event for display in the server GUI. 

This class encapsulates the data for a single log entry, including a timestamp, a type (e.\+g., "{}\+INFO"{}, "{}\+ERROR"{}), and a description. It uses Java\+FX properties to allow easy binding to UI components like Table\+View.
