\doxysubsection{Detailed Description}
A simple multi-\/threaded server for handling client connections. 

This class sets up a {\ttfamily Server\+Socket} to listen for incoming client connections on a specified port. For each connection, it creates a new {\ttfamily \doxylink{classcom_1_1example_1_1mailserver_1_1network_1_1_client_handler}{Client\+Handler}} and submits it to a thread pool for concurrent processing. \begin{DoxyRefDesc}{Deprecated}
\item[\mbox{\hyperlink{deprecated__deprecated000001}{Deprecated}}]This class is not actively used in the main application. The server logic is handled directly within the {\ttfamily \doxylink{classcom_1_1example_1_1mailserver_1_1_mail_server}{com.\+example.\+mailserver.\+Mail\+Server}} class. \end{DoxyRefDesc}
