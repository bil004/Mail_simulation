\doxysubsection{Detailed Description}
A multithreaded server for handling client connections. 

This class sets up a {\ttfamily Server\+Socket} to listen for incoming client connections on a specified port. For each connection, it creates a new {\ttfamily \doxylink{classcom_1_1example_1_1mailserver_1_1network_1_1_client_handler}{Client\+Handler}} and submits it to a thread pool for concurrent processing.
